# Catalyst 1200040 milestone 2 evidence

Proposal: [Blaze: Fully Featured Ledger Emulator, Transaction Builder for the Chang Hard Fork and PlutusV3](https://milestones.projectcatalyst.io/projects/1200040)

Current Statement of Milestones: `soms.id=10029`, Milestone 2, "Core Emulator Implementation".

## What M2 claims

M2 requires:

- A functional TypeScript prototype of core emulator features.
- Initial support for Chang governance features.
- Basic PlutusV3 capabilities in the emulator environment.
- A unit test suite for core components.
- Simulation of block production and transaction processing.
- Functional voting and proposal submission.
- Unit tests covering at least 80% of core functionality.
- A controlled environment demonstration.
- Technical documentation with usable examples.
- Peer review comments from at least two senior developers.

The emulator is deterministic. Its "block production" support is the local ledger clock and block advancement used to move transactions out of the emulator mempool and trigger epoch-boundary governance processing. It is not a consensus implementation or a replacement for cardano-node.

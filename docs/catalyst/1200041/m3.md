# Catalyst 1200041 milestone 3 evidence

Proposal: [Blaze: Maintenance, Security Improvements, Testing & Assurances for the Blaze SDK](https://milestones.projectcatalyst.io/projects/1200041)

Current Statement of Milestones: `soms.id=10011`, Milestone 3, "Implement Transaction Construction Safety Mechanisms".

## What M3 claims

M3 requires:

- A comprehensive transaction-construction test suite with unit tests, e2e tests, and property-based tests via `fast-check`.
- Improved transaction-builder error messages.
- Safety features that reduce common transaction-construction mistakes.
- Typed datum/redeemer pairing for scripts, with type errors for mismatches.
- Protection against accidentally reusing completed transaction builders.
- Explicit burn and transfer APIs so value movement is intentional.
- Documentation for transaction construction and script transaction flows.
- A coverage report showing the test suite.

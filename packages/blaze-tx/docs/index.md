---
title: "@blaze-cardano/tx"
---

`@blaze-cardano/tx` contains the transaction builder used by Blaze.

Use it to add inputs, create payment outputs, lock assets at scripts, provide datums and redeemers, attach metadata, add reference scripts, and complete a balanced transaction.

Start with the [transaction building guide](./guides/transaction-building.md) for the common flows. The API reference has the full method list for `TxBuilder`, `Value`, and the coin selectors.

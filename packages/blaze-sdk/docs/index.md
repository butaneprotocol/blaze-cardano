# SDK

`@blaze-cardano/sdk` is the main application entrypoint for Blaze. It re-exports the provider, wallet, transaction-building, and core Cardano types most applications need.

Start with these guides:

- [Developer onboarding](./guides/developer-onboarding.md) for provider setup, wallets, payments, queries, and typed Aiken contract transactions.
- [Aiken contract tutorial](./guides/aiken-contract-tutorial.md) for a step-by-step validator transaction flow.
- [Transaction building](/tx/guides/transaction-building) for payments, metadata, minting, burning, script reference outputs, and submission.
- [Script transactions](/tx/guides/script-transactions) for typed datums and redeemers.
- [Provider routing](/query/guides/provider-routing) and [provider internals](/query/guides/provider-internals) for Blockfrost, Maestro, Kupmios, routing, debug events, and backend request behavior.

For generated API references, use the package API pages from the sidebar.
